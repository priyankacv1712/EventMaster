# EventMaster – Sports Mega Event Preparation Tracker

EventMaster is a Python-based management system designed to track the preparation of large-scale sports events like the Commonwealth Games 2030.

## ✅ Features
- Modern Tkinter Dashboard UI
- Login System (Admin, Coordinator, Staff)
- Venue Manager
- Volunteer Hub with Auto IDs (VOL-2030-XXXX)
- Event Manager
- Milestone & Budget Tracking
- Reports Center
- JSON File Persistence
- Optional QR Code Generation

## ✅ Tech Stack
- Python
- Tkinter
- OOP Concepts
- JSON File Handling
